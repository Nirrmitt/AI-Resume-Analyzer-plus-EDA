🤖 AI Resume Analyzer

AI-powered Resume Screening using NLP, Semantic Matching, and Intelligent Scoring.

This project analyzes resumes against job descriptions using advanced AI techniques.

Features:

  • Supports PDF, DOCX, TXT formats
  • Semantic similarity via SentenceTransformers
  • NLP-based skill and experience extraction
  • Weighted scoring system
  • Streamlit web UI
